1.Difference between Flex-Row and Flex-Column:

In CSS,Flex box has a property called flex-direction.there are 4 possible values for flex-direction:row,row-reverse,column,column-reverse.
flex-row: When we choose flex-row, then all child items will be placed in horizantal manner.By default all child items are placed in horizantal manner only.
flex-column:when we choose column, then all child items will be placed in vertical manner.

(i) flex-direction:row; 
  
  
  
(ii) flex-direction:column;
  
  
  
2.Inline,Internal,External CSS:

Inline CSS: Inline css requires the style attribute placed inside an html element i.e..in opening tag.
Internal CSS: Internal css requires the style element placed inside the head section of a HTML file.
External CSS: External css as name says,an external .css file is created and the file is linked to .html file. It requires the link element placed inside the head section of an HTML file. Inline CSS

3.Justify-Content:

In CSS, Justify-Content property defines how the browser distributes the space between and around the content items along the main axis of the flex container and inline axis of a grid container.

4.Difference between relative and absolute Positioning:

position:relative; works based on its current position without changing the layout.i.e..,places an element relative to its current position.

Position:absolute; works based on its parent's position.It changes its position according to parent position.i.e..,places an element relative to its parent position and changing the layout around it.

5.grid-template-column:

In grids we have a property called grid-template-column.It specifies the number of columns(i.e.,No.of widths) in a grid container/layout.It takes values in pixels(px),percentages(%),fractions(fr)etc..

6.z-index:

In CSS,z-index is a property that is used to determine the stack level of an elemnet.Elements having higher index value will be placed on the top of the element having lower index value.It takes values in positive as well as in negative.It sets the order of a positioned element and its items. Large z-index value covers/overlaps with the lower z-index value.
Note:- z-index will only works on an element whose position is set to either relative,absolute,fixed.

7.Difference between margin and padding:

In CSS, margin is the space around an element's border,while padding is the space between an element's border and element's content.Margin property controls the space outside an element and the padding property controls the space inside an element.
